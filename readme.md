# Overview

# Setup
1. See my_settings.py
2. Create a file called ai.json with your settings in it

## Sample ai.json file
```
{
	"HUGGING_FACE_KEY": "",
	"HUGGINGFACEHUB_API_TOKEN": "",
	"OPENAI_API_KEY": "",
	"SERPAPI_API_KEY": "",
	"PINECONE_API_KEY": "",
	"PINECONE_API_ENV": "",
	"PINECONE_API_KEY_2": "",
	"PINECONE_API_ENV_2": ""
}
```
